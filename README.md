#  Personal Portfolio Website

Welcome to my **Personal Portfolio Website**! This project is a modern, responsive portfolio that showcases my work, skills, achievements, personal life, and professional experience. Designed with **performance, accessibility, and interactivity** in mind, it ensures a seamless user experience across all devices.

- **Responsive Design:** Looks great on mobile, tablet, and desktop.

## üöÄ Installation & Setup

### Installation Steps
Follow these steps to run the project locally on your machine:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/bigmonterss/self-portfolio-project-2025.git
    ```

2. **Navigate to the project folder:**
    ```bash
    cd self-portfolio-project-2025
    ```

3. **Install Dependencies:**
    ```bash
    npm install
    ```

4. **Start the development server:**
    ```bash
    npm run dev
    ```

### Optional: Built for Production
```bash
npm run build
```

### Deploy GitHub Pages
```bash
npm run deploy
```
